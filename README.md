# NotebookLM-Investimento-Benjamim-Graham
Metodologia de Investimentos de Benjamim Graham
Criei um caderno temático no NotebookLM sobre investimentos baseados nos ensinamentos de Benjamin Graham, reunindo conteúdos de estudo a partir de links de vídeos do YouTube e links de sites especializados. Esse material foi organizado para facilitar o aprendizado, permitindo a criação de resumos, análise dos principais conceitos e desenvolvimento de um guia de estudo estruturado sobre os fundamentos do investimento inteligente e a filosofia de investimento em valor.
